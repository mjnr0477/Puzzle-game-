

PuzzleGame

A 3x3 sliding puzzle game built with React Native and Expo.
Arrange the tiles in order to solve the puzzle and track your moves and time!

Features

Fully functional sliding puzzle

Move counter and timer

Reset button to restart the game

Works on Android and iOS with Expo Go

Ready to build APK for publishing




Installation

1. Clone the repository:



git clone https://github.com/YOUR_USERNAME/PuzzleGame.git
cd PuzzleGame

2. Install dependencies:



npm install

3. Run the project in Expo:



expo start

Scan the QR code in Expo Go to play on your phone




Build for Android / iOS

1. Install EAS CLI if not installed:



npm install -g eas-cli
eas login

2. Build APK for Android:



eas build --platform android

Download the APK → upload to Itch.io or share



Folder Structure

PuzzleGame/
├─ App.js          # Main game code
├─ package.json    # Dependencies & scripts
├─ app.json        # Expo configuration
├─ assets/         # Icons and splash images
